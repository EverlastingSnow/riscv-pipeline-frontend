# Tab + 可停靠面板系统 - 任务清单

## 阶段一：基础架构搭建

- [ ] **Task 1.1**: 创建面板状态管理（usePanelStore）
  - 定义面板状态类型和接口
  - 实现面板添加、删除、激活、折叠等方法

- [ ] **Task 1.2**: 创建 PanelTabBar 组件
  - 实现 Tab 标签显示

  - 支持最多 4 个 Tab 限制
  - 添加左右位置配置
  - **验收标准**: Tab 切换流畅，样式符合设计规范

- [ ] **Task 1.3**: 创建 DockingPanel 组件
  - 实现面板容器布局
  - 支持展开/折叠功能
  - 支持关闭按钮和最小化按钮
  - 添加内容插槽
  - **验收标准**: 面板操作响应正常，动画流畅

- [ ] **Task 1.4**: 创建 PanelContainer 组件
  - 整合 PanelTabBar 和 DockingPanel

  - 管理多个面板的显示/隐藏
  - **验收标准**: 容器完整可用，支持多面板管理

- [ ] **Task 1.5**: 弹窗系统兼容性验证（无代码改动，仅验证）
  - 确认弹窗使用 Teleport 渲染到 body
  - 确认弹窗 z-index 高于面板系统
  - 验证 IF/ID, ID/EX, EX/MEM, MEM/WB Pipeline Register 弹窗正常工作
  - 验证其他弹窗组件不受影响
  - **验收标准**: 所有弹窗功能完全不受影响
